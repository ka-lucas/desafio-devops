📘 Desafio DevOps - Documentação

Este projeto contém uma API Flask containerizada com Docker, provisionada com Terraform e publicada no Google Cloud Run, com CI/CD automatizado via GitHub Actions.

📁 Estrutura do Repositório

  • src/: Código-fonte da API
  • tests/: Testes unitários
  • terraform/homolog e terraform/prod: Infraestrutura como código
  • .github/workflows: Pipeline CI/CD
  • docs/: Documentação estática

🚀 CI/CD Pipeline

O fluxo de CI/CD executa automaticamente os seguintes passos:

  1. Verificação de qualidade: flake8, black, pylint, bandit, mypy
  2. Execução de testes com pytest
  3. Build e push da imagem Docker para o GCR
  4. Deploy via Terraform para o ambiente homologação ou produção

⚙️ Como rodar localmente

# Flask
export NAME=SeuNome
poetry run python src/app.py

# Gunicorn
poetry run gunicorn wsgi:app --bind 0.0.0.0:8080

# Docker
docker build -t desafio-api .
docker run -p 8080:8080 -e NAME=SeuNome desafio-api

📦 Terraform

export GOOGLE_APPLICATION_CREDENTIALS="path/to/credentials.json"

# Init
terraform -chdir=terraform/homolog init -reconfigure

# Plan
terraform -chdir=terraform/homolog plan

# Apply
terraform -chdir=terraform/homolog apply -auto-approve
